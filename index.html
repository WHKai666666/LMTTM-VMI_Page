LMTTM-VMI: Linked Memory Token Turing Machine for 3D Volumetric Medical Image Classification

Abstract

Leveraging neural networks, particularly Convolutional Neural Networks (CNNs), has advanced medical diagnosis via 2D biomedical image recognition. However, analyzing 3D volumetric medical images with temporal features remains challenging. We introduce Linked Memory Token Turing Machine (LMTTM), a novel neural network model that utilizes external linked memory. This model efficiently processes spatial-temporal dependencies in 3D volumetric medical images, aiding in accurate diagnoses. In experiments on six sub-datasets of MedMNIST v2, LMTTM achieved state-of-the-art(SOTA) performance in 3D image classification, highlighting its effectiveness in medical diagnosis. The code is released at:LMTTM-VMI
